Historical Release Files / Ιστορικά Αρχεία Εκδόσεων
====================================================

These manifests, validation files, and review notes belong to their original
Pass 96–98 archives. They are preserved for history and must not be used to
validate Pass 99, because the current database, application, documentation,
and search index have changed.

Αυτά τα manifest, αρχεία επαλήθευσης και σημειώσεις ελέγχου ανήκουν στα
αρχικά αρχεία των Εκδόσεων 96–98. Διατηρούνται για ιστορικούς λόγους και δεν
πρέπει να χρησιμοποιούνται για την επαλήθευση της Έκδοσης 99, επειδή η
τρέχουσα βάση, η εφαρμογή, η τεκμηρίωση και το ευρετήριο αναζήτησης έχουν
αλλάξει.
